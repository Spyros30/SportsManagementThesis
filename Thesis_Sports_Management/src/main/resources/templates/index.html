Sports Management

Welcome to athens2football, a web application that presents active football tournaments in Athens, Greece.



This is the first web application that decided not only to map all the official football tournaments in Athens, but to also give the users a chance to participate to any tournament they wish.

Built with Spring-Boot

This web application has been built by making use of Spring Boot, which is a Java Framework.Spring Boot provides a good platform for Java developers to develop a stand-alone and production-grade spring application. Also various Spring frameworks have been also used, such as Spring-MVC, Spring-Security and Spring-Data.

Thymeleaf and Bootstrap

In order to create the front-end partial of the application Thymeleaf and Bootstrap has been used. Thymeleaf is a modern server-side Java template engine for both web and standalone environments, while Bootstrap is a free and open-source CSS framework for developing responsive and mobile-first websites.

Here is the cool thing about athens2football

Athens2football is a home for every businessman who would like to host a tournament on our platform. All you have to do is send us an email and we will look into it and after reaching into a possible agreement your tournament will be hosted on our platform. Then any visitor that creates an account can join any tournament of his preference after creating his team.

Also if you are a developer and would like to be a part of our team, since we have administrators to manage the tournaments and perceive the teams send us an email, so you can have a chance to get hired and become a member of athens2football. In addition to that we are looking into developing the web application to support more sports, such as basketball, in the near future, so possible position openings could be occuring.
